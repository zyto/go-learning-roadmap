# Наработки для следующего урока

## Day 03 — break / continue / вложенные циклы

### Теория

* break
* continue
* вложенные циклы
* паттерн фильтрации через continue

### На что обратить внимание

* разница между break и continue
* контроль вложенных циклов (`i`, `j`)
* читаемость кода при вложенности

### Практика

1. остановка цикла по условию (`break`)
2. пропуск отдельных значений (`continue`)
3. перебор пар через вложенные циклы
4. вывод прямоугольника из `*`
5. вывод диапазона с исключением нескольких чисел
