# 🎯 Цель

Перейти из 1С-разработчика в Go backend разработчика и пройти собеседование на уровень **junior → middle** за 3–6 месяцев через практику.

---

# 🧠 Текущий уровень

## Сильные стороны

* Опыт разработки (синьор 1С)
* SQL (JOIN, GROUP BY)
* Понимание бизнес-логики
* Базовый опыт API

## Слабые стороны

* Go с нуля
* Слабое понимание памяти
* Нет опыта конкурентности в Go
* Поверхностный Git/Docker/Linux

# 🖥️ Среда разработки

* ОС: Windows
* Используется: WSL2 (Linux внутри Windows)
* IDE: Cursor (в режиме WSL)
* Рабочая директория: `~/go-projects/go-learning-roadmap`

## Правила среды

* Код пишется и запускается в WSL
* Git используется из WSL
* Файлы не редактируются напрямую из Windows
