# 📘 Go | Day 02 — switch + for

---

## 📖 Теория

### 1. switch (альтернатива if-else)

В Go `switch` используется вместо длинных цепочек `if-else`.

```go
switch value {
case 1:
    fmt.Println("Один")
case 2:
    fmt.Println("Два")
default:
    fmt.Println("Другое")
}
```

### Особенности:

* `break` не нужен (выход происходит автоматически)
* можно указывать несколько значений в `case`
* можно использовать `switch` без аргумента

---

### Пример с несколькими значениями

```go
day := 6

switch day {
case 6, 7:
    fmt.Println("Выходной")
default:
    fmt.Println("Рабочий день")
}
```

---

### switch без аргумента

```go
age := 20

switch {
case age < 18:
    fmt.Println("Молодой")
case age < 60:
    fmt.Println("Взрослый")
default:
    fmt.Println("Пожилой")
}
```

---

## 🔁 2. Цикл for

В Go есть только один цикл — `for`.

### Классический вариант

```go
for i := 0; i < 5; i++ {
    fmt.Println(i)
}
```

---

### Как while

```go
i := 0

for i < 5 {
    fmt.Println(i)
    i++
}
```

---

### Бесконечный цикл

```go
for {
    fmt.Println("Бесконечно")
}
```

---

### break и continue

```go
for i := 0; i < 10; i++ {
    if i == 5 {
        break
    }

    if i%2 == 0 {
        continue
    }

    fmt.Println(i)
}
```

---

## ⚠️ Важные моменты

* в Go нет `while` и `do-while`
* `switch` не падает вниз как в C/PHP
* `for` заменяет все циклы
* `switch` без аргумента — мощная альтернатива `if-else`

---

## 🧠 Сравнение с PHP

| Концепция    | Go                 | PHP             |
| ------------ | ------------------ | --------------- |
| switch break | не нужен           | обязателен      |
| while        | нет                | есть            |
| for          | один универсальный | несколько видов |
| fallthrough  | вручную            | по умолчанию    |
